# Ember.js

## Ember CLI

```bash
# Create a new project
$ ember new <my-new-app>

# Generates an application structure in the current directory
$ ember init

# Runs a specific generator
$ ember generate <generator-name> <options>
# Example: to generate controller sidebar.js from sidebar.hbs, run: ember generate controller sidebar

# To see available generators, run
$ ember help generate
```
